Connect to GitHub

Your data lives in data.json in your repo. These credentials are saved only in this browser's localStorage — never committed to GitHub.

Create a token:
GitHub → Settings → Developer settings → Personal access tokens → Fine-grained tokens → Generate new token

Repository access: Only selected repositories → pick this repo
Permissions → Contents: Read and write

Classic token also works — just needs the repo scope.
